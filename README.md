# mars_rover
Show images from Mars Rovers Curiosity, Opportunity, and Spirit, using the NASA Open API.

## Nasa Open API Information
https://api.nasa.gov/

## Prerequisites
Python3

## Setup Instructions
1. Install flask
```
pip install flask
```

2. Copy the program and html files to a directory

3. Navigate to the directory

4. Start the server
```
python3 -m main
```
